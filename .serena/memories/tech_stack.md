# Stack

- Frontend: Next.js 16.2.9, React 19, TypeScript 5, Tailwind CSS 4, NextAuth beta, Zustand; npm and pnpm lockfiles both exist.
- Backend: Java 17, Spring Boot 3.1.2, Spring Security, MyBatis 3.0.0, MySQL; Maven wrapper ./mvnw.
- Backend API base defaults to http://localhost:8080 and is configurable with NEXT_PUBLIC_API_URL.
